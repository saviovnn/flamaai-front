# FlamaAI Frontend

Interface web da plataforma FlamaAI para análise de risco de fogo.

## Tecnologias

- **Vue.js** - Framework JavaScript progressivo
- **Vite** - Build tool e dev server
- **Tailwind CSS** - Framework CSS utilitário
- **Pinia** - Gerenciamento de estado

## Pré-requisitos

- Node.js (v18 ou superior)
- pnpm

## Instalação

```bash
pnpm install
```

## Desenvolvimento

```bash
pnpm run dev
```

## Build

```bash
pnpm run build
```
